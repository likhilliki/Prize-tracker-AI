Build a FULL working project called "PriceTracker AI".

Goal:
Create an autonomous web agent application powered by the TinyFish Web Agent API that performs real browser tasks.

The application must:

1. Visit multiple e-commerce websites
2. Search for a product
3. Extract product prices
4. Compare prices across sites
5. Generate a comparison report

The project should demonstrate real web navigation, multi-step browsing, and dynamic page interaction.

Use the following stack:

Backend:
Python
FastAPI

Frontend:
Simple HTML + CSS + JavaScript

Agent:
TinyFish Web Agent API
